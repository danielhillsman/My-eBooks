  ## Installation
  1) Clone repo with:
  ````
  git clone https://github.com/danielhillsman/My-eBooks.git
  ````
  2) Run:
  ````
  cd My-eBooks/
  ````
  3) Run:
  ````
  npm i
  ````
  4) Run:
  ````
  npm run install
  ````
  5) Run:
  ````
  npm run develop

  ````
  to start the server.
